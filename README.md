Grammar for the Batak language

```
<S> ::= <subject> <verb> <noun>
<subject> ::= au | ho | ibana
<verb> ::= manuhor | maniop | mangalului | manggadis 
<noun> ::= pinahan | sira | miyak | lem | tas
```
